Projects

  • Campbelltown Council provides an API for consuming data related to the region. Their API site is here.

    The below example project walks through connection, extraction, analysis and visualisation of their local business dataset provided by the API, and delivered interactively through Power BI. The analysis begins as exploratory, and the output is a simple report which considers questions which target different business needs - for example, one of the main conclusions from a marketing perspective is that most businesses have some kind of web presence, however a few (around 3%) have no presence and may benefit from being approached regarding this.

    Note: further verification on the few businesses without a web presence would be required (e.g. a quick Google search) before actioning.
    From the OpenData API, there are a few limits on freely accessible data, one of which is the number of times downloaded.

    • Extract content

      Extract content from the API (version 2). On inspection, the API is limited to serving a maximum of 100 records per call - pagination will be required. Use List.Generate() or List.Accumulate() to iterate through the dataset. 

                                                              
let
Source =
List.Generate(
    () => // set the initial function to return a record containing the counter and request data
    [
    counter = 0,
    call = 
    Json.Document(
        Web.Contents( // request can be parametrised
        "https://www.campbelltownopendataportal.com.au/api/v2",
        [
        RelativePath = "catalog/datasets/" & dataset & "/records",
        Query = 
        [
        limit = recPerReq, // Parameters can be changed from the service with minimal effort
        apikey = apiKey // credentials are secured
        ]
        ]
        )
        )
        ],

      Multiple stop conditions may be set - the one used here is recordLimit <> null or "check the variable recordLimit is not empty". 

                                                                          
each 
if recordLimit <> null then [counter] < List.Min({total_count, Number.FromText(recordLimit)}) // get subset if set
else [counter] < total_count, // whole set of records

      Paginate via incrementing the counter and making requests with Web.Contents(). Use the Function.InvokeAfter() method to implement a delay between calls 

                                                                          
each 
[
counter = [counter] + Number.FromText(recPerReq), // parameter
call = 
        Function.InvokeAfter(
            ()=> 
            Json.Document(
                Web.Contents(
                    // urlAPIEndpoint,
                    "https://www.campbelltownopendataportal.com.au/api/v2",
                    [
                    RelativePath = "catalog/datasets/" & dataset & "/records", // parameter
                    // RelativePath = relativePath, // parameter
                    Query = 
                    [
                    limit = recPerReq,
                    offset = Text.From(counter),
                    apikey = apiKey
                    ]
                    ]
                    )
                    ), #duration(0,0,0,5)
                    )
                    ]
                    ),
    • Transform

      Now that the data has been extracted into the Power Query Editor, it can be manipulated into the final form for reporting. The result returned from the above steps is a List of Records. This can be converted to a table with Table.FromList(): 

      #"Converted to Table1" = Table.FromList(Source, Splitter.SplitByNothing(), null, null, ExtraValues.Error),

      Expand the resulting record column to new rows with Table.ExpandRecordColumn(): 
      #"Expanded Column2" = Table.ExpandRecordColumn(#"Converted to Table1", "Column1", {"counter", "call"}, {"counter", "call"}),

      The request data column (containing Records) can then be expanded to new rows using the same function: 
      #"Expanded call" = Table.ExpandRecordColumn(#"Expanded Column2", "call", {"total_count", "links", "records"}, {"total_count", "links", "records"}),

      The data records themselves may be extracted through expanding the list records using a combination of Table.ExpandListColumn() and Table.ExpandRecordColumn(): 
      #"Expanded records" = Table.ExpandListColumn(#"Expanded call", "records"),
#"Expanded records1" = Table.ExpandRecordColumn(#"Expanded records", "records", {"record"}, {"record"}),
#"Expanded record1" = Table.ExpandRecordColumn(#"Expanded records1", "record", {"id", "timestamp", "size", "fields"}, {"id", "timestamp", "size", "fields"}),
#"Expanded fields1" = Table.ExpandRecordColumn(#"Expanded record1", "fields", Record.FieldNames(#"Expanded record1"{0}[fields]), Record.FieldNames(#"Expanded record1"{0}[fields]))
